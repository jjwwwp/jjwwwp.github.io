    • 관리자: 상품 목록(상품 수정/삭제), 할인 상품 목록(상품 추가/수정/삭제), 상품 카테고리(메인/서브 카테고리 추가/수정/삭제), 관리자 목록(권한에 따라 타 관리자 추가/수정/삭제), 전체 고객 주문 목록(주문상태 변경), 문의 목록(답변 추가/수정) 접근 가능
    • 고객: 상품 목록 조회, 장바구니, 상품 주문(결제 시 포인트 사용 가능), 주문 내역 조회(주문상태에 따라 주문취소/구매확정/리뷰작성 가능), 문의 작성, 내 정보 조회/수정 가능
    • 비로그인 상태: 상품 목록 조회, 장바구니 이용 가능(상품 담겨있을 경우 로그인 시 해당 계정 장바구니로 상품 정보 이동)
  • 담당 기능: 장바구니, 결제, 전체 고객 주문 목록 관리(관리자), 나의 주문 목록 관리(고객), 상품 카테고리 목록 관리
    • 장바구니
      • 장바구니에 상품을 담으면 화면 우측 상단 장바구니 아이콘에 담긴 상품 종류의 개수가 표시됩니다.
      • 장바구니에 목록 페이지에서 상품 이미지, 상품 이름, 가격, 수량, 금액을 확인할 수 있습니다.
      • 장바구니 목록 페이지에서 상품 수량 변경, 상품 삭제, 주문이 가능합니다.
      • Javascript를 활용하여 주문할 상품을 체크박스에 표시했는지 확인하도록 했습니다.
      • 비로그인 상태에서도 장바구니를 사용할 수 있습니다.
      • 비로그인 상태에서는 장바구니 목록 조회 시 상품 삭제만 가능합니다.
      • 장바구니에 상품이 담긴 상태에서 로그인 시 해당 고객 계정의 장바구니에 상품 정보가 저장됩니다.
    • 결제
      • 상품 상세 페이지에서 바로 구매하거나, 장바구니에 담은 상품을 주문할 수 있습니다.
      • 주문 화면에서 가장 최근에 등록한 배송지가 자동으로 입력됩니다.
      • 보유 포인트를 사용할 수 있으며, JavaScript를 활용하여 결제 시 사용할 수 있는 포인트 유효값을 체크합니다.
      • 결제 버튼 클릭 시 재고량 이하를 주문한 경우 결제완료 화면이 출력되고, 메인 화면 또는 나의 주문내역 화면으로 이동할 수 있습니다.
      • 결제 버튼 클릭 시 주문 수량만큼 해당 상품의 재고가 감소합니다.
    • 전체 고객 주문 목록 관리(관리자)
      • 관리자는 전체 고객의 주문 목록을 최신순으로 조회합니다.
      • 고객 id, 주문상태별, 월별 검색이 가능합니다.
      • 결제완료 상태에서 버튼을 클릭할 수 있고, 버튼 클릭 시 주문상태가 배송완료로 변경됩니다.
    • 나의 주문 목록 관리(고객)
      • 주문코드, 상품 이름, 주문상태, 가격, 수량, 사용포인트, 적립포인트 등을 표시하였습니다.
      • 기타옵션에서 주문상태에 따라 주문취소/구매확정/리뷰작성 버튼을 클릭할 수 있게 하였습니다.
      • 결제완료 상태에서 주문취소 버튼 클릭 시 사용 포인트가 취소되고, 주문상태가 주문취소로 변경됩니다.
      • 배송완료 상태에서 구매확정 버튼 클릭 시 포인트가 적립되고, 주문상태가 구매확정으로 변경됩니다.
      • 구매확정 상태에서 리뷰작성 버튼 클릭 시 리뷰작성 페이지로 이동하고, 리뷰작성 완료 후 리뷰작성 버튼이 보이지 않게 하였습니다.
    • 상품 카테고리 목록 관리
      • 관리자는 상품 카테고리 목록을 조회할 수 있습니다.
      • 메인 카테고리 목록에서 카테고리명을 클릭하면 서브 카테고리 목록으로 이동합니다.
      • 메인 카테고리 목록에서 새 카테고리(메인, 서브 카테고리 쌍을 입력)를 입력할 수 있습니다.
      • 서브 카테고리 목록에서 카테고리를 수정/삭제할 수 있습니다. 서브 카테고리에 상품이 존재할 경우 수정/삭제를 할 수 없다는 알림창이 출력됩니다.

      • 로그인 및 회원가입을 할 수 있습니다.
